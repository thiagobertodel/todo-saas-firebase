# 🚀 To-Do SaaS

Aplicação de tarefas com autenticação Google e banco de dados na nuvem usando Firebase.
![Uploading Captura de Tela 2026-03-17 às 21.45.05.png…]()



## ✨ Funcionalidades
- Login com Google
- CRUD de tarefas
- Edição de tarefa
- Prioridade
- Filtro
- Deploy no Netlify
- Persistência com Firestore
- Regras de segurança por usuário
