# 🚀 To-Do SaaS

Aplicação de tarefas com autenticação Google e banco de dados na nuvem usando Firebase.

## ✨ Funcionalidades
- Login com Google
- CRUD de tarefas
- Edição de tarefa
- Prioridade
- Filtro
- Deploy no Netlify
- Persistência com Firestore
- Regras de segurança por usuário

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firestore
- Netlify
